# DX12
项目中使用了《DirectX 12 3D游戏开发实战》也就是龙书所给出的代码框架.  并且担心不同功能间相互影响, 整合在一起恐怕复杂度剧增, 自己水平不够难以掌握, 因此各个功能都是独立开来分别完成的.


# **各个功能的简要展示**  

1.基于几何着色器的曲面细分  

2.图像与几何两种方式的边缘检测

3.基于法线贴图与位移贴图的波浪模拟  

4.PCSS  

[5.延迟渲染](#延迟渲染)

[6.PBR](#PBR)
